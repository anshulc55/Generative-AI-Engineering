# Generative AI Engineer Mastery  
### LLM Engineering • RAG • LoRA / QLoRA • Agentic AI

This repository contains the **official codebase** for the course  
**Generative AI Engineer Mastery: LLM, RAG & Agentic AI**.

This is **not a collection of random notebooks or demos**.  
It is a **structured, production-oriented learning repository** designed to help you think and work like a **real Generative AI Engineer**.

---

## 🚀 What This Repository Is

This repository is a **guided engineering workspace** that supports the course by providing:

- Clean, readable, and production-inspired code
- Incremental builds instead of one-off scripts
- Multiple approaches to the same problem (trade-offs matter)
- Reusable modules instead of throwaway code
- Realistic AI system architectures

You will **not start from a blank screen**.  
You will learn by **reading, modifying, extending, and reasoning about real systems**.

---

## ❌ What This Repository Is NOT

- ❌ Not a prompt collection  
- ❌ Not copy-paste snippets without context  
- ❌ Not a “magic wrapper” around APIs  
- ❌ Not framework hype  

This repo exists to teach **engineering judgment**, not shortcuts.

---

## 🧠 How to Use This Repository (Very Important)

### The correct way to learn from this repo:

1. **Watch the lecture first**
   - Understand *why* the system exists
2. **Read the code**
   - Before running it
3. **Run the code**
   - Observe behavior, cost, latency, failures
4. **Modify the code**
   - Change models, parameters, chunking, prompts
5. **Break the system**
   - This is where real learning happens

If you only run the code without thinking, you will miss the point.
